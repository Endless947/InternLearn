Platforms       : Android, iOS, Web, Windows, Linux, macOS

================================================================================
1. OVERVIEW
================================================================================

InternLearn is a cross-platform learning application built with Flutter and
Supabase. Learners navigate structured educational content in this hierarchy:

  Subject -> Chapter -> Topic -> Subtopic -> Slides

Supported slide types:

  1. Content Slides (markdown + key points)
  2. MCQ Slides (single-correct multiple choice with explanation)
  3. Match Slides (pair matching interaction)

================================================================================
2. CURRENT IMPLEMENTATION STATUS
================================================================================

A) Authentication
- Email/password sign up
- Email/password login
- Session-based routing with AuthGate
- Logout from profile tab

B) Learning Content
- Subject/chapter/topic/subtopic drill-down flow
- Slide viewer with merged ordered sequence across slide types
- Progress UI and gated interaction completion for MCQ/Match

C) Profile and Profile-Related Features
- Profile tab now shows:
  - Display name
  - Email
  - Bio
  - Member since
  - Short user id
- Manage Profile page is functional:
  - Edit display name and bio
  - Persist changes using Supabase auth user metadata
- Theme Settings page is functional:
  - Light/Dark/System selection
  - App-wide theme updates through Riverpod provider
  - Preference persisted in user metadata
- Notifications Settings page is functional:
  - Email digest, new content, and reminders toggles
  - Preference persisted in user metadata
- Danger Zone is implemented:
  - Delete account request confirmation dialog
  - Records deletion request metadata
  - Signs out user immediately
